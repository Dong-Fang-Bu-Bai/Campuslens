# 第四周 V3：M1、M2、M4、M5 实施记录

日期：2026-06-08

本记录对应第四周构造阶段计划中 M1、M2、M4、M5 的工程实施范围。V3 在 V2 上传检索、地标详情、地图联动和反馈落库基础上，补充个人历史、打卡留言、反馈校正样本闭环、高并发配置口径和前端体验增强。

## M1 图片上传与检索主流程

新增 `GET /api/me/search-records`，登录用户可查看自己的检索历史。接口只按 Bearer token 解析出的 `userId` 查询 `search_record`，不接受前端传入用户编号，避免普通用户读取他人记录。返回内容包括上传图路径、最高候选、状态、提示信息、最近反馈状态和 Top-5 快照。

后端配置补充 Tomcat 线程、请求队列、Hikari 连接池、算法调用连接/读取超时和算法校正通知线程池。该配置用于体现本机单体部署的并发扩展能力，不宣称生产级集群能力。

## M2 地标图像库与元数据

新增 `check_in`、`check_in_like`、`check_in_reply` 和 `correction_sample` 表。打卡留言复用 L01-L10 地标数据和地图百分比坐标，不新增独立地点字典。校正样本表保存来源反馈、检索记录、上传图路径、确认地标、Top-5 快照和算法同步状态。

正式地标样本库与校正样本集保持边界：管理员采纳反馈后只写入 `correction_sample`，不会直接把用户上传图加入 `datasets/landmarks/` 或正式样本表。

## M4 结果展示与地图导览

前端新增个人历史页面和校园打卡留言板。登录用户可从个人历史查看服务端检索记录，并从历史 Top-5 候选继续发起反馈。打卡留言板支持发布打卡、点赞和一级回复，游客使用 `guest#number` 展示。

结果区新增滚轮循环切换和上一项/下一项控制。页面偏好新增主题、背景和中英文轻量切换，状态保存到 `localStorage`。管理员后台增加反馈详情区，展示上传图、Top-5 快照、校正样本同步状态和算法采纳建议。

## M5 反馈与记录统计

新增 `GET /api/admin/feedback/{id}` 返回反馈详情。管理员将反馈状态更新为 `accepted` 时，后端创建 `correction_sample` 并异步通知算法服务；状态更新为 `ignored` 时只更新反馈状态，不生成校正样本。

算法通知成功后，校正样本状态写为 `synced`，并保存 `suggestAccept`、`reviewScore`、`reason`、`sarEligible` 和 `nextAction`。算法不可用或超时时，校正样本保留为 `sync_failed`，管理员采纳动作不回滚。

## 最小 M3 支撑

算法服务新增 `POST /api/v1/adaptation/correction-samples`。该接口接收校正样本元数据，将记录追加到 `correction_samples.jsonl`，并基于 Top-5 经验分构造伪概率评估 `reviewScore`。当前实现只体现“基于 SAR 思路的可靠样本过滤与采纳建议”，不宣称完整复现 SAR 论文源码。

## 验证记录

- 后端：`mvn test` 通过，18 项测试；覆盖任务终态轮询、游客令牌、所有者作用域幂等、反馈状态与所有权限制。
- 算法：D 盘 GPU 环境执行 `python -m pytest`，15 项测试全部通过，覆盖批量顺序、坏图隔离、OOM 拆批及单项隔离、设备选择和健康信息。
- 前端：`npm run build` 通过，Vite 成功生成生产构建。
- 运行与并发：2026-06-10 完成 MySQL 全链路启动和分层高压测试，详细指标、瓶颈结论与数据归零结果见 [运行、回归与并发测试记录](12_runtime_and_concurrency_test.md)。

## GPU 与异步队列增量

在 V3 功能基线上新增 Redis 异步检索任务和 RTX 4060 GPU 推理。MySQL `search_record` 保存任务状态、所有者作用域幂等摘要、令牌哈希、尝试次数、`next_attempt_at` 和 lease；Redis 使用活跃 Set、ready List、processing/delayed ZSet 完成容量准入、带凭证领取和延迟重试。前端提交后按 1 秒、2 秒、5 秒退避轮询，并在 `localStorage` 保存未完成任务。

真实验收结果与容量、Redis 故障、lease 恢复记录见 [GPU 与 Redis 异步队列测试记录](13_gpu_async_queue_test.md)。
