# 项目总览

## 项目定位

CampusLens（校园慧眼）是校园地标智能检索与导览系统。用户上传校园照片后，系统返回 Top-5 候选地标，并提供地标详情、代表图片、静态地图位置和反馈入口。

## 已确定范围

范围内：

- 用户端图片上传与检索。
- Top-5 候选地标结果展示。
- 地标详情与校园平面图静态标注。
- 用户识别结果反馈。
- 地标资料、样本图片和检索记录的基础维护。
- 后端接口、算法服务、数据库和本地图片/向量文件存储。

范围外：

- 实时导航和室内导航。
- 自训练图像模型。
- Web 与小程序双端并行。
- 社交打卡等非核心功能。
- 完整复杂运营后台。

## 技术栈

| 层次 | 技术口径 |
| --- | --- |
| 前端 | Vue |
| 后端 | Spring Boot |
| 算法服务 | Python FastAPI |
| 数据库 | MySQL |
| 检索 | DINOv2 提取图像特征，按地标样本特征估计均值和协方差，使用马氏距离与卡方置信度评分 |
| 地图 | 校园平面图静态标注 |

## 五周里程碑

| 周次 | 阶段 | 核心目标 |
| --- | --- | --- |
| 第 1 周 | 初始阶段 | 项目计划、需求规约、原型、环境、接口和数据规范 |
